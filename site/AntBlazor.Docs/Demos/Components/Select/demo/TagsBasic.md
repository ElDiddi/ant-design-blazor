## zh-CN

The mode `Tags` does not differ in its basic functions from the mode `Multiple`. If the user is allowed to create his own tags, the `Tags` mode should be used in conjunction with `AllowCustomTags`.


## en-US

The mode `Tags` does not differ in its basic functions from the mode `Multiple`. If the user is allowed to create his own tags, the `Tags` mode should be used in conjunction with `AllowCustomTags`.
